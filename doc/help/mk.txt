Usage: mk [task...]

  Runs tasks defined in a task file. Searches for mkdoc.js in the current
  directory (or parent directories) and executes the specified tasks.

Options
  -f, --file=[FILE...]    Load specific task files
  --tasks                 Print task list
  -h, --help              Display help and exit
  --version               Print the version and exit
