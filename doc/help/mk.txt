mk

Task runner.

Usage: mk [task...]

  Runs tasks defined in a task file. Searches for mkdoc.js in the current
  directory or parent directory and executes the tasks specified on the command
  line.

Options
  -f, --file=[FILE...]    -f, --file=[FILE...] Load specific task files
  --tasks                 --tasks Print task list
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit

  Report bugs to https://github.com/mkdoc/mktask/issues

mktask@1.3.7
