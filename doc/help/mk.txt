Usage: mk [task...]

  Task runner.

Options
  -f, --file=[FILE...]    -f, --file=[FILE...] Load specific task files
  --tasks                 --tasks Print task list
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit
