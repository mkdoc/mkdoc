mkabs

Usage: mkabs [options]

  Make relative links absolute.

Options
  -b, --base=[URL]        -b, --base=[URL] Base URL for absolute links
  -r, --relative=[PATH]   -r, --relative=[PATH] Relative path when repository
                          url
  -g, --greedy            -g, --greedy Convert links starting with # and ?
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit

  Report bugs to https://github.com/mkdoc/mkabs/issues

mkabs@1.2.2
