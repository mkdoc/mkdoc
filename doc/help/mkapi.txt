mkapi

Usage: mkapi [options] [files...]

  Documentation generator.

Options
  -o, --output=[FILE]     -o, --output=[FILE] Write output to FILE (default:
                          stdout)
  -t, --title=[VAL]       -t, --title=[VAL] Title for initial heading
  -l, --level=[NUM]       -l, --level=[NUM] Initial heading level (default: 1)
  -L, --lang=[LANG]       -L, --lang=[LANG] Language for fenced code blocks
                          (default: javascript)
  -i, --indent=[NUM]      -i, --indent=[NUM] Number of spaces for JSON
                          (default: 2)
  -a, --ast               -a, --ast Print AST as JSON
  --[no]-private          --[no]-private Enable or disable private symbols
  --[no]-protected        --[no]-protected Enable or disable protected symbols
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit

  Report bugs to https://github.com/mkdoc/mkapi/issues

mkapi@1.1.7
