mkcat

Reads markdown documents.

Usage: mkcat [options] [files...]

  Reads markdown documents from stdin and files; writes serialized nodes to
  stdout.

  Reads markdown documents from stdin and files; writes serialized nodes to
  stdout.  The output is newline-delimited JSON records representing the abstract syntax
  tree for the input document(s).

Options
  -h, --help              Display help and exit
  --version               Print the version and exit

  Report bugs to https://github.com/mkdoc/mkcat/issues

mkcat@1.1.5
