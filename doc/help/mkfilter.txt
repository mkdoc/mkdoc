mkfilter

Usage: mkfilter [options]

  Filter nodes by type.

Options
  -i, --invert            -i, --invert Invert the filter
  --document              --document Filter document nodes
  --softbreak             --softbreak Filter softbreak nodes
  --linebreak             --linebreak Filter linebreak nodes
  --list                  --list Filter list nodes
  --item                  --item Filter item nodes
  --paragraph             --paragraph Filter paragraph nodes
  --text                  --text Filter text nodes
  --heading               --heading Filter heading nodes
  --emph                  --emph Filter emph nodes
  --strong                --strong Filter strong nodes
  --link                  --link Filter link nodes
  --image                 --image Filter image nodes
  --code                  --code Filter code nodes
  --code-block            --code-block Filter code block nodes
  --block-quote           --block-quote Filter block quote nodes
  --thematic-break        --thematic-break Filter thematic break nodes
  --html-inline           --html-inline Filter html inline nodes
  --html-block            --html-block Filter html block nodes
  --custom-inline         --custom-inline Filter custom inline nodes
  --custom-block          --custom-block Filter custom block nodes
  --eof                   --eof Filter eof nodes
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit

  Report bugs to https://github.com/mkdoc/mkfilter/issues

mkfilter@1.1.3
