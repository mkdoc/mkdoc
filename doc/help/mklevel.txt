mklevel

Modify heading levels.

Usage: mklevel [options]

Options
  -1=[NUM]                -1=[NUM] Modify level 1 headings by NUM
  -2=[NUM]                -2=[NUM] Modify level 2 headings by NUM
  -3=[NUM]                -3=[NUM] Modify level 3 headings by NUM
  -4=[NUM]                -4=[NUM] Modify level 4 headings by NUM
  -5=[NUM]                -5=[NUM] Modify level 5 headings by NUM
  -6=[NUM]                -6=[NUM] Modify level 6 headings by NUM
  -a, --all=[NUM]         -a, --all=[NUM] Modify all headings by NUM
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit

  Report bugs to https://github.com/mkdoc/mklevel/issues

mklevel@1.0.3
