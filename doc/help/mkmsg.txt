Usage: mkmsg [-ph] [--prepend] [--help] [--version] [--message=<msg>]

  Message injection.

Options
  -m, --message=[MSG]     Custom message, parsed as markdown
  -p, --prepend           Prepend message to the stream
  -h, --help              Display help and exit
  --version               Print the version and exit
