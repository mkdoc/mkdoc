mkmsg

Usage: mkmsg [options]

  Message injection.

Options
  -m, --message=[MSG]     -m, --message=[MSG] Custom message, parsed as markdown
  -p, --prepend           -p, --prepend Prepend message to the stream
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit

  Report bugs to https://github.com/mkdoc/mkmsg/issues

mkmsg@1.2.5
