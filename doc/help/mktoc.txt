mktoc

Usage: mktoc [options]

  Generates a table of contents index.

Options
  -t, --title=[TITLE]     -t, --title=[TITLE] Set initial heading
  -l, --level=[NUM]       -l, --level=[NUM] Set level for initial heading
  -d, --depth=[LEVEL]     -d, --depth=[LEVEL] Ignore headings below LEVEL
  -m, --max=[LEVEL]       -m, --max=[LEVEL] Ignore headings above LEVEL
  -p, --prefix=[VAL]      -p, --prefix=[VAL] Set link destination prefix to VAL
  -b, --base=[URL]        -b, --base=[URL] Base URL for absolute links
  -B, --bullet=[CHAR]     -B, --bullet=[CHAR] Character for bullet lists
  -E, --delimiter=[CHAR]  -E, --delimiter=[CHAR] Delimiter for ordered lists
  -D, --disable           -D, --disable Disable automatic links
  -o, --ordered           -o, --ordered Create an ordered list
  -s, --standalone        -s, --standalone Standalone index, discards input
  -h, --help              -h, --help Display help and exit
  --version               --version Print the version and exit

  Report bugs to https://github.com/mkdoc/mktoc/issues

mktoc@1.0.8
